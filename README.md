# IDEA codcut plugin

[IDEA](https://www.jetbrains.com/) platform codcut plugin. Only tested on Intellij.

# Usage

Go to `File > Settings > Codcut` and set your token there. Then highlight some code and press `Ctrl + Alt + U`!
